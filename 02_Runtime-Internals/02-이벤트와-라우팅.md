# 이벤트와 라우팅

## 이 문서의 질문

- 이벤트는 어떻게 정규화되는가?
- `session.*`와 `agent.*`는 어떤 관계인가?
- router는 어떻게 0..N delivery를 만든다고 볼 수 있나?

## 원본 소스 포인터

- `clawhip/docs/native-event-contract.md`

## 핵심 이해 포인트

- canonical event는 `session.*` 계열을 중심으로 본다.
- legacy `agent.*`는 호환성 계층으로 유지된다.
- route filter는 message text보다 structured metadata 중심으로 설계한다.
